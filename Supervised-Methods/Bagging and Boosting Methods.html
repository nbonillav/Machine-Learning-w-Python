Ejercicio 1¶

[50 puntos] En este ejercicio se usará la tabla de datos students_dropout.csv. Este conjunto de datos contiene datos de una institución de educación superior sobre diversas variables relacionadas con los estudiantes de pregrado, para investigar el impacto de estos factores en la deserción de los estudiantes y el éxito académico. La tabla contiene 3630 filas y 13 columnas, las cuales se explican a continuación.

  • marital_status: Estado Civil. (Categórica)
  • daytime_evening_attendance: Si el estudiante asiste a clases durante el día o la noche. (Categórica)
  • educational_special_needs: Si el estudiante tiene alguna necesidad educativa especial. (Categórica)
  • debtor: Si el estudiante es deudor. (Categórica)
  • tuition_fees_up_to_date: Si las tasas de matrícula del estudiante están al día. (Categórica)
  • gender: Género del estudiante. (Categórica)
  • scholarship : Si el estudiante tiene beca. (Categórica)
  • age_at_enrollment: Edad del estudiante al momento de la inscripción. (Numérica)
  • international: Si el estudiante es extranjero. (Categórica)
  • unemployment_rate: Tasa de desempleo. (Numérica)
  • inflation : Tasa de inación. (Numérica)
  • gdp: PIB per cápita. (Numérica)
  • target: Indica si el estudiante se graduó o abandonó. (Categórica, Variable a predecir)

Realice lo siguiente:

  • a) Cargue la tabla de datos students_dropout.csv correctamente y verifique que no tenga datos nulos o ausentes.
  • b) Use Bosques Aleatorios, ADABoosting, GradientBoosting y XGBoosting en Python para generar modelos predictivos para la tabla datos students_dropout.csv usando el 80% de los datos para la tabla aprendizaje y un 20% para la tabla testing.   • d) Construya un DataFrame que compare los mejores modelos construidos en los items anteriores para la tabla students_dropout.csv. Para esto en cada una de las filas debe aparecer un modelo predictivo y en las columnas aparezcan los Indices Precisión Global, Error Global, Precisión Positiva (PP) y Precisión Negativa (PN). Cuál de los modelos es mejor para estos datos?

def asCategoryType(df, asCategory=True):
    df_copy = df.copy()
    target_var = df_copy['target']
    df_copy = df_copy.drop(columns=('target'))
    if asCategory:
        num_df = df_copy.select_dtypes(include=[np.number])
        cat_df = df_copy.select_dtypes(exclude=[np.number])

    if not cat_df.empty:
        for i in cat_df:
            cat_df[i] = cat_df[i].astype('category')
            df_copy = pd.concat([num_df, cat_df, target_var], axis=1)
    else:
        df_copy = pd.concat([num_df, cat_df], axis=1)
    
    return df_copy

def indices_general(MC, nombres = None):
    precision_global = np.sum(MC.diagonal()) / np.sum(MC)
    error_global     = 1 - precision_global
    precision_categoria  = pd.DataFrame(MC.diagonal()/np.sum(MC,axis = 1)).T
    if nombres!=None:
        precision_categoria.columns = nombres

    dictionary = {
        "Matriz de Confusión":MC, 
        "Precisión Global": precision_global, 
        "Error Global": error_global, 
        "Precisión por categoría": precision_categoria
        }
    
    return dictionary

def clasification(matriz_confusion):
    VN = matriz_confusion[0][0]
    FP = matriz_confusion[0][1]
    FN = matriz_confusion[1][0]
    VP = matriz_confusion[1][1]

    precision_global = (VN+VP)/(VN+FP+FN+VP)
    error_global = 1 - precision_global
    precision_positiva = VP/(FN+VP)
    precision_negativa = VN/(VN+FP)
    
    diccionario = {'precision_global':precision_global,
                    'error_global': error_global,
                    'precision_positiva_Dropout': precision_positiva,
                    'precision_negativa_Graduate': precision_negativa}
    
    return diccionario

class ConsensoPropio:
    def __init__(self):
        self._modelos = {}
        self._precisiones = {}
        self._label_encoder = LabelEncoder()
    
    def fit(self, X_train, y_train):
        n = len(X_train)
        
        # Convertir etiquetas para XGBoost
        y_train_encoded = self._label_encoder.fit_transform(y_train)
        
        preprocesamiento = ColumnTransformer(
            transformers=[
                ('num', StandardScaler(), ['age_at_enrollment', 'unemployment_rate', 'inflation_rate', 'gdp']) 
            ]
        )
        arbol = DecisionTreeClassifier(max_depth=15)
        # Lista de los 4 tipos de modelos que vamos a usar
        tipos_modelos = [
            ('KNeighborsClassifier', Pipeline([
                ('preprocesador', preprocesamiento),
                ('clasificador', KNeighborsClassifier(n_neighbors=6, metric="euclidean", weights='uniform'))
            ]), y_train),  # AÑADIR y_train como tercer elemento
            
            ('DecisionTreeClassifier', Pipeline([
                ('preprocesador', preprocesamiento),
                ('clasificador', DecisionTreeClassifier(criterion='entropy', max_depth=3, min_samples_split=2, min_samples_leaf=2))
            ]), y_train),  # AÑADIR y_train como tercer elemento
            
            ('XGBoostClassifier', Pipeline([
                ('preprocesador', preprocesamiento),
                ('clasificador', XGBClassifier(n_estimators = 20, learning_rate = 1.0, max_depth = 15))
            ]), y_train_encoded),  # AÑADIR y_train_encoded como tercer elemento
            
            ('AdaBoostClassifier', Pipeline([
                ('preprocesador', preprocesamiento),
                ('clasificador', AdaBoostClassifier(n_estimators = 20, learning_rate = 1.0, estimator = arbol))
            ]), y_train)  # AÑADIR y_train como tercer elemento
        ]
        
        for i, (nombre, modelo, y_train) in enumerate(tipos_modelos, 1):
            print(f"Entrenando modelo: {nombre}")
            
            # 1. Generar muestra bootstrap
            indices = np.random.choice(range(n), size=n, replace=True)
            X_bootstrap = X_train.iloc[indices]
            
            # Usar los datos y apropiados para cada modelo
            if nombre == 'XGBoostClassifier':
                y_bootstrap = y_train[indices]  # y_train_encoded (array numpy)
            else:
                y_bootstrap = y_train.iloc[indices]  # y_train original (Series pandas)
            
            # 2. Entrenar el modelo
            modelo_entrenado = modelo.fit(X_bootstrap, y_bootstrap)
            
            # 3. Calcular precisión global
            predicciones = modelo_entrenado.predict(X_bootstrap)
            precision_global = accuracy_score(y_bootstrap, predicciones)
            
            # 4. Guardar modelo y precisión
            clave_modelo = f'modelo{i}'
            clave_precision = f'PG{i}'
            
            self._modelos[clave_modelo] = modelo_entrenado
            self._precisiones[clave_precision] = precision_global
            
            print(f"Precisión global {clave_precision}: {precision_global:.4f}\n")
        
        # print("Resumen de precisiones:")
        # for clave, precision in self._precisiones.items():
        #     print(f"  {clave}: {precision:.4f}")
    
    def __str__(self):
        return f"ConsensoPropio con {len(self._modelos)} modelos entrenados"

class ConsensoPropio:
#constructor
    def __init__(self):
        self._modelos = {}
        self._precisiones = {}
        self._label_encoder = LabelEncoder()
    
    def fit(self, X_train, y_train):
        n = len(X_train)
        
        # Convertir etiquetas ['target'] para XGBoost
        y_train_encoded = self._label_encoder.fit_transform(y_train)
        
        preprocesamiento = ColumnTransformer(
            transformers=[
                # ('cat', OneHotEncoder(), ['marital_status', 'daytime_evening_attendance', 'educational_special_needs', 'debtor', 'tuition_fees_up_to_date', 'gender', 'scholarship_holder', 'international']),
                ('num', StandardScaler(), ['age_at_enrollment', 'unemployment_rate', 'inflation_rate', 'gdp']) 
            ]
        )
        
        # Lista de los modelos que vamos a usar
        # Parametros según metodos anteriores
        arbol = DecisionTreeClassifier(max_depth=15)
        tipos_modelos = [
            ('KNeighborsClassifier', Pipeline([
                ('preprocesador', preprocesamiento),
                ('clasificador', KNeighborsClassifier(n_neighbors=6, metric="euclidean", weights='uniform'))
            ]), y_train),
            
            ('DecisionTreeClassifier', Pipeline([
                ('preprocesador', preprocesamiento),
                ('clasificador', DecisionTreeClassifier(criterion='entropy', max_depth=3, min_samples_split=2, min_samples_leaf=2))
            ]), y_train),
            
            ('XGBoostClassifier', Pipeline([
                ('preprocesador', preprocesamiento),
                ('clasificador', XGBClassifier(n_estimators = 20, learning_rate = 1.0, max_depth = 15))
            ]), y_train_encoded),
            
            ('AdaBoostClassifier', Pipeline([
                ('preprocesador', preprocesamiento),
                ('clasificador', AdaBoostClassifier(n_estimators = 20, learning_rate = 1.0, estimator = arbol))
            ]), y_train)
        ]
        
        # nombre del modelo, tipo de modelo y el y_train este cambia para XGBoost
        for i, (nombre, modelo, y_train) in enumerate(tipos_modelos, 1):
            print(f"Entrenando modelo: {nombre}")
            
            # Muestra bootstrap
            indices = np.random.choice(range(n), size=n, replace=True)
            X_bootstrap = X_train.iloc[indices]
            
            if nombre == 'XGBoostClassifier':
                y_bootstrap = y_train[indices]  # y_train_encoded
            else:
                y_bootstrap = y_train.iloc[indices]  # y_train original
            
            # Entrenando el modelo
            modelo_entrenado = modelo.fit(X_bootstrap, y_bootstrap)
            
            # Calculo de precisión global
            predicciones = modelo_entrenado.predict(X_bootstrap)
            precision_global = accuracy_score(y_bootstrap, predicciones)
            
            # guardando modelo y precisión segun el modelo
            clave_modelo = f'modelo{i}'
            clave_precision = f'PG{i}'
            
            self._modelos[clave_modelo] = modelo_entrenado
            self._precisiones[clave_precision] = precision_global
            
            print(f"Precisión global {clave_precision}: {precision_global:.4f}\n")
                
    def predict(self, X_test):
    # Calculando los pesos
        suma_precisiones = sum(self._precisiones.values())
        pesos = {}
        for i in range(1, 5):
            clave_precision = f'PG{i}'
            pesos[clave_precision] = self._precisiones[clave_precision] / suma_precisiones
        
        # Obtener las probabilidades de cada modelo para todo X_test
        probabilidades_modelos = {}
        
        for i in range(1, 5):
            clave_modelo = f'modelo{i}'
            modelo = self._modelos[clave_modelo]
            
            # Para TODOS los modelos obtener probabilidades
            probas = modelo.predict_proba(X_test)
            probabilidades_modelos[clave_modelo] = probas
        
        # Para cada fila en X_test, aplicar la fórmula de consenso
        predicciones_finales = []
        n_filas = len(X_test)
        
        for i in range(n_filas):
            # Diccionario para acumular lo ponderados por clase
            suma_ponderada_clases = {}
            
            # Para cada clase, calcular la suma ponderada
            for clase in range(len(self._label_encoder.classes_)):
                suma_clase = 0.0
                
                # Sumar las contribuciones de los 4 modelos
                for j in range(1, 5):
                    clave_modelo = f'modelo{j}'
                    clave_precision = f'PG{j}'
                    
                    proba_modelo = probabilidades_modelos[clave_modelo][i][clase]
                    peso_modelo = pesos[clave_precision]
                    
                    suma_clase += peso_modelo * proba_modelo
                
                suma_ponderada_clases[clase] = suma_clase
            
            # Encontrar la clase con la suma máxima
            clase_ganadora = max(suma_ponderada_clases, key=suma_ponderada_clases.get)
            
            # convertir target para que no haya problemas en comparaciones
            prediccion_final = self._label_encoder.inverse_transform([clase_ganadora])[0]
            predicciones_finales.append(prediccion_final)
        
        return np.array(predicciones_finales)
    
    
    def precision_global(self, X_test, y_test):
        y_pred = self.predict(X_test)
        return accuracy_score(y_test, y_pred)
    
    def error_global(self, X_test, y_test):
        precision = self.precision_global(X_test, y_test)
        return 1 - precision
    
    def precision_por_clases(self, X_test, y_test):
        y_pred = self.predict(X_test)
        return classification_report(y_test, y_pred, output_dict=True)

# print
    
    def __str__(self):
        return f"ConsensoPropio con {len(self._modelos)} modelos entrenados"
